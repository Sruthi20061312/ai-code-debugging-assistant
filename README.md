# AI Code Debugging Assistant 🚀

## 📌 Problem Statement
Developers spend significant time debugging and optimizing code manually.

## 💡 Proposed Solution
An AI-powered assistant that:
- Detects errors in Python and SQL code
- Suggests corrected code
- Explains mistakes clearly
- Recommends optimizations

## 🛠 Tech Stack
- Python
- Streamlit
- OpenAI API
- dotenv

## 🔬 Technology Readiness Level (TRL)
TRL 3 – Experimental Proof of Concept validated through testing.
